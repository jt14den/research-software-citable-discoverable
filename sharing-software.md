## Introduction

Hosting your code in a public version-controlled repository increases its visibility, ensures transparency, and allows others to cite or build on your work. GitHub is a widely-used platform for sharing research software.

Repositories with README files, clear names, and project structure are easier to find and reuse.

:::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::: instructor

If learners are new to GitHub, show examples of well-documented repos first. Encourage participants to navigate to a sample repo.

## Challenge 1: What Makes a Repository Useful?

Visit a GitHub repository for a research project. What information is helpful? What is missing?

:::::::::::::::::::::::: solution

Helpful elements may include: README file, license, list of contributors, installation instructions, and usage examples. Missing items might be unclear documentation or no stated license.
