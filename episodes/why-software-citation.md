## Introduction

Software plays a critical role in research but has historically been under-recognized. Treating software as a citable output ensures proper attribution, improves transparency, and supports reproducibility.

Citing software with DOIs and formal metadata allows others to reference specific versions and gives creators academic credit.

:::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::::: instructor

Ask participants if they've ever reused software without citing it, or struggled to find or cite code used in a paper. This helps them connect to the motivation.
